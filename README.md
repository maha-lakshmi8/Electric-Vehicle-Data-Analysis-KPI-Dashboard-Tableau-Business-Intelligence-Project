# Electric-Vehicle-Data-Analysis-KPI-Dashboard-Tableau-Business-Intelligence-Project
Electric Vehicle Data Analysis Dashboard using Tableau to track EV growth trends, state-wise adoption, BEV vs PHEV comparison, and key market insights through interactive visualizations.

📌Project Overview

This project presents an interactive Electric Vehicle (EV) Data Analysis Dashboard built using Tableau. The dashboard analyzes EV adoption trends, vehicle growth patterns, manufacturer performance, CAFV eligibility distribution, and state-wise market penetration.

The objective is to transform raw EV data into actionable insights for business decision-making.

🎯 Business Problem

With the rapid growth of electric vehicles, stakeholders require:

Clear visibility of EV adoption trends

BEV vs PHEV market comparison

State-wise distribution analysis

Manufacturer performance insights

CAFV eligibility classification tracking

This dashboard centralizes these insights in one interactive view.

📊 Key Performance Indicators (KPIs)

Total Vehicles

Average Electric Range

Total BEV Vehicles

Total PHEV Vehicles

% Distribution of EV Types

These KPIs measure market size and performance.

📈 Dashboard Features

✔ Model Year Trend Analysis
✔ State-wise EV Distribution
✔ Top 10 Manufacturers
✔ BEV vs PHEV Comparison
✔ CAFV Eligibility Breakdown
✔ Interactive Filters

📍 Key Insights

BEV vehicles dominate the EV market.

EV adoption has increased significantly in recent years.

A few manufacturers contribute to a major share of registrations.
